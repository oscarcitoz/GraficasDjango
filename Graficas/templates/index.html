BIENVENIDO AL SISTEMA

     Esta aplicación consiste en elaborar una gráfica(proyección) con los datos que se encuentran en los archivos del sistema.

     En el menú principal se muestran diferentes opciones, antes que nada debe cargar su archivo para poder visualizarlo en la aplicación, a continuación se presentan unos lineamientos para que tenga noción de como utilizar el sistema:

    

  1. Solo se admiten 2 tipos de archivos: csv y txt.
    2. 

  2. Asegurece antes de realizar cualquier operación haber copiado los archivos en la carpeta del servidor.
    3. 

  3. Para que el archivo sea registrado debe hacer click en cargar,seleccionar el tipo y completar el formulario que se muestra.(Asegurece de ver la imagen que se encuentra en la sección cargar, ya que esta es un ejemplo de como debe ser la estructura del archivo).
    4. 

  4. Si ocurre algun error inesperado el sistema se redireccionara automáticamente a una página en la cual se muestran los posibles errores.
    5. 

  5. Ver: En esta opción se selecciona que tipo de archivos queremos ver, al elegir, se cargara una ventana con los archivos que se encuentran almacenados y se podrán realizar funciones tales como:

      

    • Ver el archivo: Se refleja en una tabla el archivo guardado.
      • 

    • Realizar Gráfica: Se realiza la proyección con los datos del archivo seleccionado.
      • 

    • Eliminar:Se borra de la base de datos el archivo seleccionado.
      •
